刚刚使用git，现将流程及简单命令进行整理记录

## Ubuuntu16.04 + git + github ? 
-----
1 下载git 
  apt-get install git
2 注册github帐号
  建立代码仓库（我的为skyshu.github.io）
3 在需要建立仓库的地方（本地）
  git init
4 建立和github的关联
  git remote add origin https://github.com/skyshu/skyshu.github.io.git
  （不能忘填 .git）
5 更新本地仓库
  git pull origin master
6 将修改文件进行添加
  git add .
7 提交
  git commit
8 提交到github服务器
  git push -u origin master
